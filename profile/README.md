# AI 기반 장애인 취업 공고 추천 서비스 HOW

## 프로젝트 소개

이 프로젝트는 AI를 활용하여 장애인들에게 맞춤형 취업 공고 추천과 취업과 관련된 커뮤니티 활동이 가능한 서비스입니다.

## 주요 기능

- **서비스 소개**: 서비스의 목적과 사용 방법에 대한 상세한 안내
- **맞춤형 취업공고 제공**: AI를 활용한 개인별 맞춤 취업 로드맵 생성
- **커뮤니티**: 취업과 관련된 커뮤니티 활동 가능
- **마이페이지**: 개인 정보 관리 및 설정
